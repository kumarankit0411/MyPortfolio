I contribute to socli as a core contributor. Socli is a command line interface for stack overflow and provides you with the ability to view answers to any question in command line. DuckDuckGo is a meta search engine that does not collect any information about the user searching a query on it. This makes it an anonymous search engine that most users prefer if they do not to want themselves to be tracked on internet. Aside from this, DDG is great for programmers as they get instant answers to most programming queries and I have made several instant answers for DDG such as java programming cheatsheet, urllib cheatsheet. Have a look at one of my IA's that is triggered with the query "urllib cheatsheet" on DuckDuckGo
